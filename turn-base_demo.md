# Design

* 核心玩法
  * 数值系统
    * 护盾
      * 降低至零后不会立刻死亡，而是反向增长压力
      * 护盾被穿透会导致较大的副作用：1）累计压力 2）一段时间无法获得护盾 3）行动力下降？
    * 压力值
      *  压力值满直接进入**死门（death blow check）**
      *  高压状态会带来其他副作用：遇到心魔BOSS？数值弱化？不可控行为？ 
    * AP：行动点
    * Damage：倍率
  * 流派构筑：基于精神分析理论：防御机制
    * 评价维度
      * 行动力
      * 输出数值
      * 生存
        * 防御数值
        * 压力恢复
      * 其他
        * 资源能力
    * 流派设计
      * Intellectualization：玻璃大炮 / 弱恢复能力 / 行动力枯竭
      * Projection：强行动力 / 恢复能力强 / 
      * Denial：
      * 癫狂（？）：
      * Repression：
      * Sublimation
      * humour
  * 怪物设计
    * 学校：考验输出数值能力？
    * 原生家庭：考验生存能力
    * 强权：最终大考
